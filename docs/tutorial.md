# Tutorial

ChatPharo provides an interactive tutorial designed to introduce features progressively.

## Starting the tutorial

- World menu: **ChatPharo → Tutorial**
- Or evaluate: `ChatPharoTutorial start`

## Setup wizard

A setup wizard is available from the world menu to configure:

- user profile hints (user level, preferences)
- agent selection
- recommended defaults via the Advisor

## Suggested learning path

1. Run the tutorial once.
2. Enable one skill (e.g., Spec2 or Roassal).
3. Try a tool-assisted question (e.g., “Find methods that send a network request”).
4. Enable Memory to keep stable preferences.
5. Enable Multivers for complex tasks.
